Ghost Turnstile Server

Local CAPTCHA solving server for Cloudflare Turnstile challenges

Server Active on port 8888

API Endpoints

GET /solve?sitekey={key}&url={url}

Create a solving session and display the Turnstile widget

GET /status?session={id}

Check the status of a solving session

GET /test

Test page with demo Turnstile widget

POST /token

Internal endpoint for token submission

Quick Start

Example URLs:

http://localhost:8888/solve?sitekey=1x00000000000000000000AA&url=https://example.com

Note: Use one of these demo sitekeys for testing:

  • 1x00000000000000000000AA - Always passes (visible widget)
  • 2x00000000000000000000AB - Always blocks
  • 3x00000000000000000000FF - Always passes (invisible)
